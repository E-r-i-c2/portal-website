Learn to Code on Cursor

Welcome! This step-by-step guide will help you create your very first project and learn the basics of coding using Cursor. No experience neededâ€”just follow along!

Step 1: What is Cursor?

Cursor is a modern code editor that makes it easy to start coding, experiment, and build projects. You can write code, run it, and see results instantly.

Step 2: Create Your First Project

  1. Open Cursor on your computer.
  2. Click File > New Project (or use the + New button if available).
  3. Choose a folder for your project and give it a name (e.g., my-first-website).

Step 3: Make Your First File

  1. Right-click in the file explorer and select New File.
  2. Name it index.html.
  3. Copy and paste this code into your new file:
<!DOCTYPE html>
<html>
  <head>
    <title>My First Web Page</title>
  </head>
  <body>
    <h1>Hello, world!</h1>
    <p>This is my first web page on Cursor.</p>
  </body>
</html>

Step 4: View Your Web Page

  1. Right-click index.html and select Open with Live Server (or use the built-in preview feature).
  2. You should see your web page in a new tab!

Step 5: Add Some Style

  1. In the same folder, create a new file called style.css.
  2. Add this code to style.css:
body {
  background: #18181c;
  color: #e5e5e5;
  font-family: sans-serif;
  padding: 40px;
}
h1 {
  color: #e11d48;
}
  1. Link your CSS file in index.html by adding this line inside the <head>:
<link rel="stylesheet" href="style.css">

Step 6: Make It Interactive

  1. Create a new file called script.js.
  2. Add this code to script.js:
alert('Welcome to your first website!');
  1. Link your JS file in index.html just before </body>:
<script src="script.js"></script>

Step 7: Keep Going!

  • Try changing the text, colors, or adding more elements.
  • Look up more HTML, CSS, and JavaScript tutorials online.
  • Ask for help on the Portal Support page or join our community!

Great job! You've just made your first website. Keep experimenting and building!
